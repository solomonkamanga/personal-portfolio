Bw Events

Android Application Development

A mobile app that primarily showcases events that are occurring in Botswana. It was initially developed with the intent to disseminate information about events that were organized as part of the celebration of Botswana's 50th Independence but the focus then shifted to including all events taking place in the country at any given time. The current focus now allows for more growth in the usage of the application for years to come. The application is a hybrid application as it leverages native android components such as the WebView with HTML, CSS and JavaScript.
